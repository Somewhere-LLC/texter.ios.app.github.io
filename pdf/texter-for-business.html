Features
主要機能
用途に応じて最適なAIエンジンを選択。録音から共有まで一気通貫で対応します。
AI音声文字起こし
Apple標準 / AmiVoice / AWS Transcribe / Whisper の4エンジン対応。句読点も自動挿入。
リアルタイム翻訳
DeepL / Google翻訳に対応。100言語以上をサポートし、文字起こしと同時に翻訳を実行。
画像・動画OCR
撮影した資料やホワイトボードをテキストデータに変換。動画ファイルからの音声抽出にも対応。
AI要約 (GPT-4o)
長文テキストをGPT-4oが自動要約。会議の要点を素早く把握し、情報共有を効率化。
動画字幕生成 + PiP
動画に自動で字幕を生成。ピクチャ・イン・ピクチャで他アプリ使用中も字幕付き再生が可能。
無音カット
不要な無音部分を自動検出して除去。3つのプリセット（弱・標準・強）で調整可能。

Security
セキュリティ・プライバシー

完全オンデバイス処理
Apple標準エンジンはデバイス上で処理が完結。音声データがサーバーに送信されることはありません。

Whisper / GPT-4o を使用する場合のみサーバー通信が発生。エンジン選択はユーザー自身が行います。
